# Tags group component

## Description

This component is used to display tags component at the bottom of the screen.

## Classes

- tags: a flexing container to hold all tags.
- tag: a single tag.

## HTML Structure

This sass file is to be used on the following html structure:

```HTML
<div class="tags">
    <div class="tag">tag1</div>
    <div class="tag">tag2</div>
    <div class="tag">tag3</div>
</div>
```
